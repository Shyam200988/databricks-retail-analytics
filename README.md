# Databricks Retail Analytics Project

## Overview

This project demonstrates an end-to-end data engineering pipeline using Databricks Community Edition.

## Architecture

Medallion Architecture:

* Bronze Layer: Raw CSV ingestion
* Silver Layer: Data cleaning and transformations
* Gold Layer: Business KPIs and aggregations

## Tech Stack

* Databricks (Spark, Delta Lake)
* GitHub (Version Control)

## Data Pipeline Flow

CSV Files → Bronze Tables → Silver Tables → Gold KPIs

## Gold Layer Outputs

* Product Sales
* Customer Sales
* Daily Sales
* City Sales
